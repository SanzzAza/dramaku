Frequently Asked Questions

Semua yang perlu kamu ketahui tentang SanzXD API.

Apa itu SanzXD API?
SanzXD API adalah platform REST API modern yang menyediakan layanan download TikTok, Instagram, dan YouTube. Gratis, tanpa watermark, dan tanpa API key.
Apakah API ini gratis?
Ya, sepenuhnya gratis. Tidak ada biaya tersembunyi, tidak perlu registrasi, dan tidak ada batasan ketat. Langsung pakai.
Bagaimana cara integrasinya?
Sangat mudah! Cukup kirim GET request ke endpoint yang tersedia dengan parameter url berisi link video yang ingin didownload.
